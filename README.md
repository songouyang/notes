本地预览

```console
$ git clone https://github.com/songouyang/notes.git && cd notes
$ git submodule update --init --recursive
$ ln -s `pwd`/source/icarus.yml `pwd`/themes/icarus/_config.yml
$ yarnpkg
$ yarnpkg run server
```
